# o2o           
koa + mysql + react 前后分离 商城项目       

## frontend      
前端页面     
```bash
    cd frontend

    npm i 
    
    npm run dev
```

## server  
服务端     
```bash
    cd server

    npm i

    npm run dev

```

    # 建表
